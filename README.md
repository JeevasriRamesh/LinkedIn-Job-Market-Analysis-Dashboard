This project analyzes a real-world LinkedIn job postings dataset to uncover insights into skill demand, salary trends, job locations, and experience requirements. Using Python, the data was cleaned, processed, and enhanced through feature engineering techniques such as extracting skills from job descriptions and deriving experience levels. The results were visualized through an interactive Streamlit web application and a Power BI dashboard, enabling users to explore job market trends dynamically through filters and charts. The project demonstrates end-to-end data analysis, from raw data preprocessing to building interactive and insightful visualizations.


Live Demo: https://linkedin-job-market-analysis-dashboard.onrender.com
